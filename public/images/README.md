## Hur man exporterar från Figma:

1. Öppna Figma-designen: https://www.figma.com/design/FeBn9FuJrvjqlpYUwFMDSY/SITE-v.1
2. Välj det element du vill exportera
3. I högra sidopanelen, under "Export", välj lämpligt format (PNG eller JPG)
4. För PNG: Välj 2x eller 3x för högre kvalitet
5. För JPG: Välj kvalitet 90%
6. Klicka "Export [namn]"
7. Spara filen i denna mapp med rätt filnamn

## Alternativt: Använd Figma Desktop

Om du har Figma Desktop kan du också högerklicka på ett element och välja "Copy/Paste" -> "Copy as PNG" och klistra in direkt.
